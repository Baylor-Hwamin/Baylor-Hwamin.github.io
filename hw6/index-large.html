Low Vision Style changelog

  • Increased font size
    • Since most of the browsers has descent zoom function, I focused on meeting similar font size for all elements of this page.
    • Relatively small texts were magnified more than other bigger texts.
  • Changed color scheme
    • Inverted the color scheme from the “Light mode” to the “Dark mode”.
    • Background color was not set to real black color(#000000), so this feature could stress the buttons or other interactive forms.
  • Applied elements
    • Texts, Buttons, Background, and Input forms
  • What you learned from the testing of the magnifier.
    • "Looks good" UI may contain too large pictures or too big text. For example, the font size of contents and headings has significant difference. This may leads users are confused when they decide the proper zoom ratio.
    • Bold style and fat font can effectively increase the readability than the font size. In my opinion, this is because user can distinguish the fat line of the alphabet faster.
